# 可編輯交易紀錄版更新

本版新增「交易紀錄可編輯」功能：

- 每筆交易旁新增：編輯、複製、刪除。
- 編輯可修改：日期、股票代號、股票名稱、買進/賣出、股數、價格、手續費、證交稅、操作方向、進場理由、停損、停利、備註。
- 複製會把原交易帶入新增交易表單，日期預設為今天，確認後可儲存成新紀錄。
- 編輯或刪除後會透過既有 /api/state 同步到 PostgreSQL / Supabase。
- 保留原本 AI 股票分析、明日操作計畫、原先持股管理、均價試算、雲端同步等功能。

部署方式：
1. 解壓縮 ZIP。
2. 將資料夾內所有檔案覆蓋上傳到 GitHub repo。
3. Render 重新部署最新 commit。
4. DATABASE_URL 與 PYTHON_VERSION 不需要重設。
