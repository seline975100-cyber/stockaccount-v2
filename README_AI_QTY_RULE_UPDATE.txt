# AI 建議股數級距更新

本版已調整 AI 股票分析中的「建議股數」邏輯，避免低價股只建議個位數、或高價股建議過大單次交易量。

## 股價級距規則

- 股價在幾十元到幾百元：單次買進 / 賣出建議約 30～100 股
- 股價約 1000～3000 元：單次買進 / 賣出建議約 5～20 股
- 股價約 3000～7000 元：單次買進 / 賣出建議約 1～5 股
- 股價更高者：單次買進 / 賣出建議約 1～3 股

## 同步更新位置

- 明日操作計畫的「建議股數」
- 個人化持股判斷的「建議股數」
- 加碼後均價試算
- 賣出估計實現損益

若系統判斷為賣出情境，會顯示建議賣出股數區間；若為續抱 / 等待情境，會顯示「0 股；觸發買點再買 X～Y 股」。
