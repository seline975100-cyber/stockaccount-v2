本版更新：AI 股票分析已加入個人化持股判斷

新增內容：
1. 每檔 AI 分析會結合你的目前持股股數、成本均價、投資成本佔比與報酬率。
2. 會計算目前價格距離你的成本多少。
3. 會依照部位集中度調整加碼建議，不會只看股票本身訊號。
4. 會估算「如果照建議股數加碼，買後新均價會是多少」。
5. 會估算「如果賣出一半，目前約可實現多少損益」。
6. 原先持股管理、雲端同步、AI 動態新增持股功能都保留。

部署方式：
1. 解壓縮 ZIP。
2. 將資料夾內所有檔案覆蓋上傳到 GitHub。
3. Render 重新 Deploy latest commit。
4. DATABASE_URL 與 PYTHON_VERSION 不需重新設定。
