本版新增：選股篩選系統

新增位置：左側目錄「選股篩選」

功能：
1. 每次按「查詢推薦股票」都會重新評估股票池。
2. 會自動排除目前已持有股票。
3. 預設掃描一組台股候選股票，也可自行輸入股票代號作為自訂股票池。
4. 每次列出 5 檔候選股票。
5. 分析方式包含：
   - 長期事業基本面：營收、利潤率、資產負債表、現金流、負債、管理團隊、護城河。
   - 技術情境：趨勢、動能、成交量、關鍵價位。
   - 財務篩選條件：營收成長、利潤率、資產負債表、現金流。
   - 質化檢查與風險訊號。

注意：
- 財務資料或行情資料若外部來源暫時抓不到，系統會明確標示，不會假裝資料完整。
- 此功能是研究與決策輔助，不保證獲利。
