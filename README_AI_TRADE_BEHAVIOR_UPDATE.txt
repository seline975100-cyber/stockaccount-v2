# AI 股票分析：加入過往買賣行為研究

本版新增「你的過往買賣模式」分析，讓 AI 股票分析不只看持股比例，也會讀取投資復盤/交易紀錄。

新增內容：

1. 讀取原先持股與後續新增交易紀錄。
2. 分析每支股票的過往買進價位區間、平均買進價、平均買進股數。
3. 分析過往賣出價位區間、平均賣出價、平均賣出股數。
4. 判斷加碼習慣，例如逢低加碼、追價加碼、小量試單、分批佈局。
5. 判斷出售習慣，例如獲利減碼、虧損停損、成本附近調節。
6. 將過往買賣模式與明日操作計畫、具體建議保持一致。

顯示位置：

AI 股票分析 → 每支股票的「個人化持股判斷」內，新增「你的過往買賣模式｜用投資復盤校正建議」。

注意：如果某支股票尚未有足夠交易紀錄，系統會先以原先持股成本與目前持股比例判斷，之後交易紀錄越多，判斷會越個人化。
